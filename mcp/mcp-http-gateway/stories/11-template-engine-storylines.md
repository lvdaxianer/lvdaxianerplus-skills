## STORY-TE-001: 基础变量替换

### 故事目标
- **核心问题**：验证模板引擎能正确替换基础变量 `{param}`
- **目标用户**：MCP 工具开发者
- **预期产出**：变量替换成功，返回正确值

### 关键节点/里程碑
1. **起点**：传入模板 `{userId}` 和参数 `{ userId: "123" }`
2. **步骤1**：模板引擎识别变量
3. **步骤2**：查找参数值
4. **步骤3**：替换变量
5. **终点**：返回 `"123"`

### 角色/参与者
| 角色 | 职责 | 输入 | 输出 |
|------|------|------|------|
| 模板引擎 | 解析和替换变量 | 模板字符串 + 参数对象 | 替换后的字符串 |

### 数据流转
| 节点 | 输入数据 | 输出数据 | 数据传递方式 |
|------|----------|----------|--------------|
| 识别变量 | `"{userId}"` | 变量名 `userId` | 内存 |
| 查找参数 | `userId` | `"123"` | 内存 |
| 替换 | `"123"` | `"123"` | 返回 |

### 测试场景

#### 正常分支
| 条件 | 测试值 | 预期结果 |
|------|--------|----------|
| 字符串参数 | `{ userId: "abc" }` | `"abc"` |
| 数字参数 | `{ count: 100 }` | `"100"` |
| 布尔参数 | `{ enabled: true }` | `"true"` |

#### 异常分支
| 异常类型 | 触发条件 | 处理方式 |
|----------|----------|----------|
| 参数不存在 | `{ userId }` 无对应参数 | 保持原样 `{userId}` |
| 空参数 | `{ userId: null }` | `"null"` |
| 空字符串 | `{ userId: "" }` | `""` |
