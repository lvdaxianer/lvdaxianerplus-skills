# TOAST-002: Toast 错误提示显示

## 故事目标

- **核心问题**：验证 Toast 组件能正确显示错误类型提示
- **目标用户**：Dashboard 用户
- **预期产出**：操作失败后显示红色 Toast，3 秒后自动消失

## 关键节点/里程碑

- **起点**：调用 showToast(message, 'error')
- **中间步骤**：
  1. 创建 toast DOM 元素
  2. 设置 class 为 'toast toast-error'
  3. 设置背景色 #FEE2E2，文字色 #991B1B
  4. 添加到 toast-container
  5. 3 秒后自动移除
- **终点**：Toast 显示并消失
