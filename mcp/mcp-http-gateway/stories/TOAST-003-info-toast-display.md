- **核心问题**：验证 Toast 组件能正确显示信息类型提示

## 关键节点/里程碑

- **起点**：调用 showToast(message, 'info')
- **中间步骤**：
  1. 创建 toast DOM 元素
  2. 设置 class 为 'toast toast-info'
  3. 设置背景色 #DBEAFE，文字色 #1D4ED8
