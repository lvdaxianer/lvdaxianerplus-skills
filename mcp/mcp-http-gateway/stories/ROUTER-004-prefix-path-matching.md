# ROUTER-004: Prefix 前缀路径匹配

## 故事目标

- **核心问题**：验证前缀匹配能正确匹配路径开头并返回剩余路径
- **目标用户**：开发者/测试人员
- **预期产出**：路径以 prefix 开头时返回 matched: true 和 remaining 路径

## 关键节点/里程碑

- **起点**：matchPath 方法接收到 matchType = 'prefix'
- **中间步骤**：
  1. 获取 route.path prefix 字符串
  2. 执行 pathname.startsWith(prefix)
  3. 提取 remaining 路径
- **终点**：返回 { matched: true/false, params: { remaining } }
