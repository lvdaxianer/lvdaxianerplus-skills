# JSON-001: JSON 格式化按钮功能

## 故事目标

- **核心问题**：验证 JSON 格式化按钮能正确格式化输入内容
- **目标用户**：Dashboard 用户
- **预期产出**：点击按钮后 JSON 内容格式化为缩进 2 空格格式

## 关键节点/里程碑

- **起点**：用户在 textarea 输入 JSON
- **中间步骤**：
  1. 点击"格式化"按钮
  2. 触发 formatMockJson()
  3. 读取 textarea.value
  4. JSON.parse() 解析
  5. JSON.stringify(parsed, null, 2) 格式化
  6. 更新 textarea.value
- **终点**：textarea 显示格式化后的 JSON

## 角色/参与者

| 角色 | 职责 | 输入 | 输出 |
|------|------|------|------|
| formatMockJson | 格式化 JSON | textarea value | formatted JSON |
| JSON.parse | 解析 | string | object |
| JSON.stringify | 格式化 | object | formatted string |

## 流程/路径

用户点击格式化按钮
